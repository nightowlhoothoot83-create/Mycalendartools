Tide Times

Search NOAA tide prediction stations and check high and low tide times without changing the MyCalendarTools flow.

This tool uses NOAA CO-OPS public station metadata and tide prediction data. Location access is optional and only used in your browser to find nearby stations.

Start with a station search

Try a station name, US coastal city, state abbreviation, or NOAA station ID. The results will appear here.

NOAA station data

Station search uses the NOAA CO-OPS Metadata API for tide prediction stations. Tide times are pulled from NOAA predictions for the selected date.

Safety note

Tides can affect swimming, boating, fishing, crossings and coastal access. Always check local warnings and official marine forecasts before making safety decisions.

Coverage

This version supports NOAA tide prediction stations. Other regional sources can be added later using the same layout.
